# BikeShare Stations

This folder contains resources and scripts related to the analysis and visualization of BikeShare stations in the context of the Beach BIA project. The data and scripts here are focused on extracting the stations relevant to a Toronto BIA or neighbourhood, and using them for analysis.

## Contents

- **BikeShare-Stations.qmd**: A Quarto markdown file for analyzing and visualizing BikeShare station data.
- **BikeStation_SpatialFunctions.R**: An R script containing spatial functions for processing and analyzing BikeShare station data.
