# Exploratory Data Analysis (EDA)

This folder contains the initial exploration of the datasets to identify trends, outliers, and potential correlations.

## Key Files
- **TheBeachBIA-EDA.qmd**: The primary exploration document covering general visitor trends.
- **Beaches_Bikes_Combined.qmd**: Analysis focusing specifically on the intersection of weather patterns and Bike Share usage.
- **Beaches_311_Combined.qmd**: Observing 311 service request data with local activity.

## Purpose
The goal of these documents was to inform future analyses and productive outputs for the project.
